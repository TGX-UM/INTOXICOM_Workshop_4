# Report of the fourth INTOXICOM Workshop

This workshop was held in Birminham, UK, in June 2025.

## Previewing the PDF

1. go to https://preview.biohackrxiv.org/
2. copy/paste `https://github.com/BiGCAT-UM/INTOXICOM_Workshop_4/` in the "Repository:" field
3. Click the "Generate PDF" button and wait a few seconds
4. (Optionally) Download the PDF
